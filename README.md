# Imersão IA PhD | Engenharia de Sistemas Agênticos em Maceió

Bem-vindo ao repositório da Landing Page oficial da **Imersão Presencial IA PhD em Maceió**. Este projeto é uma página estática voltada para a divulgação e inscrição no evento sobre Engenharia de Sistemas Agênticos.

## 🚀 Sobre o Projeto

A **Imersão IA PhD** é um ambiente de experimentação intensivo onde as equipes constroem e implantam o primeiro funcionário agêntico do zero. A landing page foi desenvolvida com foco em performance, acessibilidade e design moderno (utilizando grid técnico, animações de scroll reveal e tipografia limpa).

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica e acessível.
- **CSS3**: Estilização, animações (Scroll Reveal), flexbox/grid e design responsivo.
- **JavaScript Vanilla**: Lógica para interatividade, animações de scroll e controle de vídeo.

## 📁 Estrutura do Projeto

```text
├── assets/          # Imagens, vídeos e logotipos
├── index.html       # Estrutura principal da página
├── script.js        # Lógica de interatividade e animações
└── styles.css       # Estilos globais e componentes
```

## 🌐 Como Rodar Localmente

Como é um site estático simples, não há necessidade de build ou configurações complexas.

1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
   ```
2. Abra o arquivo `index.html` diretamente no seu navegador, ou utilize uma extensão como o **Live Server** no VS Code para visualizar com auto-reload.

## 🚢 Como Publicar (Deploy)

Recomendamos utilizar o [Vercel](https://vercel.com) ou [GitHub Pages](https://pages.github.com) para hospedar este projeto. Basta conectar seu repositório do GitHub na plataforma escolhida e o deploy será automático a cada atualização na branch principal.
